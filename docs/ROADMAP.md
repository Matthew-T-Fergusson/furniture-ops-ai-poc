# Roadmap

- Add deterministic helper scripts for common workflow mutations.
- Add pytest/dbt-style validation tests around guardrail views.
- Add synthetic scenario fixtures for acquisition, pending sale, split payment, cancellation, and delivery completion.
- Add lightweight dashboard queries for throughput, stale inventory, gross margin, delivery-fee economics, and contractor reliability.
- Enforce a strict high-level furniture type taxonomy plus flexible secondary subtype.
- Add expired-listing detector that creates relist tasks automatically.
- Add CI to load the schema into Postgres and run validation queries.
